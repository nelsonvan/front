Sélectionner en JQuery

Le coeur de JQuery se trouve dans la section d'éléments pour ensuite éxécuter des methodes dessus.

$(selecteur) accepte en arguments tout sélecteur CSS, JQuery nous renvoie ensuite une collection d'un ou plusieurs élements DOM enveloppés dasn un object JQuery.

$('div') séléctionne tout les divs $('.titre') séléctionne tout les éléments qui on la class="titre" $('#boite') sélectionne l'unique

Cette enveloppe JQuery est celle qui donne accés a toutes les fonctionnalité de JQuery

  • Rajouter des classes
  • Modifier le contenue HTML
  • Afficher ou cacher des éléments
  • Les animer etc.

Tout les éléments séléctionnés en javascript pur (document.getElementByld, querySelectior etc) sont des éléments DOM et tous ceux sléctionnés avec $('...') sont des éléments DOM dans une enveloppe JQuery

JQuery dispose de plusieurs méthodes pour parcourir l'arbre du document (le DOM)

  • children() parcourt les enfants
  • find() parcourt les descendants
  • next() renvoie le voisin suivant
  • prev() renvoie le voisin précédent
  • parent() renvoie le parent
  • eq() renvoie, parmi une liste d'éléments, celui qui se trouve a l'index fourni
