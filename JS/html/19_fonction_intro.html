Les instructions

Un script js est un fichier contenant des instructuon, placées les unes a la suites des autres. chacune des ses instruction peut etre vue comme un ordre un ordre donné au moteur javascript, une action que vous souhaitez réaliser, a la maniére d'une recette de cusine

  • Faire fondre le beurre
  • Mélanger la farine.
  • Verser le chocolat petit a petit ext

Les instructions se suivent et sont effectuées les unes a la suite des autres, Du haut vers le bas(comme en CSS), le browser parse (analyse) dans l'ordre du flux

Chaque instruction doit se transformer par un point virgule ; afin d'indiquer au moteur javascript que cette instruiction est terminé

Les fonctions

Les fonctions permetent de regroupée une serie d'instruction pour pouvoir les réutiliser. quand on a besoin du code d'une serie d'instruction, plutot que de réécrire a chaque fois toute les instruction , on appellera une fonction par son nom (faireUngateau). cette fonction contien l'ensemble des instructions a executer.

Par example, la fonction alert est une fonction native (intégré au navigateur) qui contien en gros les instruction suivante :

  • créer le cadre de la boite de dialogue
  • posisionner le cadre dans la fenétre
  • créer un background semi transparent sur toute les page
  • placer un bouton avec le texte ok et un événement click
  • créer une zone de texte qui affichera le texte de l'art a prése,ter au visiteur

Nous on a juste besoin d'utiliser la fonction alert() en lui donnant un paramétre example alert('coucou'). La fonction éxécutera toutes les innstructions a chaque fois.
on peut utiliser cette fonction alert quand on veut en la déclenchant avec alert(quelque chose). ce "quelque chose" est un argument attendu par la fonction.

var maVariable = document.querySelector('h2'); alert('coucou');

Les fonctions querySelector et alert sont disponible dans le navigateur, ce sont des méthode

si vous voyez qulque chose qui ressemble a une variable et qui est suivie de paranthéses c'est une fonctions

Les fonctions prennent souvent des arguments - des bouts de données dont elles ont besoin pour faire leur travaille . Les arguments sont placé entre les paranthéses, séparées par des virgules s'il y en a plusieurs.

Par example la fonction alert() fait apparaitre une pop-up mais vous devez donner une chaine comme arguments pour indiquer ce que vous souhaitez écrire dans la fenétre
