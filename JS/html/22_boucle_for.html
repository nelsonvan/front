Les boucles

Les instructions for et while qui sont deux types de boucles les plus utilises en js, servent a répter plusieurs lignes de code

la boucle FOR

For prend 3 paramétres encadrés dans des paranthéses, suivie entre accolades des instruction a éxecuter autant de fois que la boucle l'indique, les paramétres sont

  • initiallisation : ici on déclare la variable quiservira de compteur
  • condition : tant que cette condition est vraie la boucle séxecutera
  • Expression finale : expression qui est éxecutée a chaque fin de boucle
for (initiallisation; condition; incrémentation){ instructuion 1; instructuion 2; }
