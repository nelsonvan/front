Angular

1. C'est quoi Angular ? : Angular est un framework front-back coté client, il est utilisé pour créer des applications web, il fait partie de la technologie 'mean'; (Mongo DB, Expresse js, AgularJS, NodeJS), il a etait créer par google

2. Pourquoi l'utiliser ?

Le developpement ess plus rapide, nous pouvons l'utiliser sur toute les plateformes (Windows) (Mac OS) (linux) de plus Angular utilise aussi le language Typescript.

3. COMPOSANTS

sont des blocks qui représnete la base d'une applications Angular = un arbre de composants qui va utiliser des decorateur pour tranformer nos classes en composants, sont composé de 3 partie du typescript du HTML et du CSS ses composants donne du sens a nos balise

4. SERVICES

importer a l'aide d'un composants nommé 'injectable', permet de transmettre des données entre nos différents composants, permet de ne pas répeter une ligne de code plusieurs fois (gain de temps).

5. Prérequis pour utiliser Angular

de bonne notion en html et en Javascript (fonctions, boucles, objects etc.) il faut égalment installé Nodejs, Git, Vscode.

6. Installer le framework

Il y a 2 maniére de l'insataller (le framework) la premiére est de le faire avec Git bach la 2 eme maniére est de l'installer avec le terminal de Vs code

methode git pour installer Angular

créer un dossier, clique droit sur git bach Here cela va ourire la console il suffira de tappez la commande a l'interieur : npm install -g @Angular/cli (cette commande permet d'installer Angular) une fois cela fait il faudra tappe une autre commande qui permmetra de créer notre premiére applicaitons, : ng new (puis le nom de votre project) Ex : ng new FirstAngular, lorsque l'installation est terminé vous pourrez voir a quoi ressemble votre fichier il suffira de l'ouvrir avec Vs code clique droit dessus open wich Vs code.
